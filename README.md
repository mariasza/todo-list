# API Todo List

## Description

```bash
API for to-do list control
```

## Installation

```bash
$ npm install
```

## Creating database

```bash
$ docker-compose up -d
$ npm run db:recreate

```

## Running the app

```bash
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
