# API Todo List

## Description

```bash
API for to-do list control
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```

## Creating database

```bash
$ docker-compose up -d
$ npm run db:recreate

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
